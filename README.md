# 📚 Lumina Library Management System

A modern, full-stack **Library Management System** designed to manage books, users, and roles efficiently with a clean UI and role-based access control.

---

## 🚀 Features

### 👤 User (Student Portal)

* View available books 📖
* Search books 🔍
* Borrow / Return books
* Personalized dashboard

### 🧑‍💼 Employee Portal

* Manage book inventory (Add / Update / Delete)
* View logged-in students
* Issue and return books

### 🛠️ Admin Portal

* Full system control 🔐
* View all users (Students + Employees)
* Remove employees
* Monitor active sessions
* System analytics

## 🔐 Role-Based Access Flow

```
Login Page → Select Role
        ├── Student → Student Portal
        ├── Employee → Employee Portal → Manage Books + View Students
        └── Admin → Admin Dashboard → Manage Employees + View Users
```
